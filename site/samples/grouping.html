This sample demonstrates subgraphs that are created only as groups are expanded.

The model is initially a random number of nodes, including some groups, in a force-directed layout. When a group is expanded, a "SubGraphExpanded" Diagram Listener calls a function to generate a random number of nodes in a layered digraph layout inside the group if it contains none already. Each non-group node added has a unique random color, and links are added by giving each node one link to another node.

The addition of nodes and links is performed within a transaction to ensure that the diagram updates itself properly. The diagram's force-directed layout and the layered digraph layouts within each group are performed again when a sub-graph is expanded or collapsed.
