Class LayoutEdge

An edge represents a link in a LayoutNetwork. It holds layout-specific data for the link.

Modifying a LayoutNetwork or a LayoutVertex or a LayoutEdge does not invalidate the Layout or raise any changed events.

Constructor Detail
LayoutEdge()
This constructs an edge that does not know about any Link.
Property Detail
{LayoutVertex} fromVertex
Gets or sets the LayoutVertex that this edge comes from. Setting this property does not change any LayoutVertex#destinationEdges collection.
{Link} link
Gets or sets the Link associated with this edge, if any. The value may be null if this edge is a "dummy" edge, not represented by a real Link in the Diagram.
{LayoutNetwork} network
Gets or sets the LayoutNetwork that owns this edge. The default value is null. It is set automatically for you in LayoutNetwork#addEdge.
{LayoutVertex} toVertex
Gets or sets the LayoutVertex that this edge goes to. Setting this property does not change any LayoutVertex#sourceEdges collection.
Method Detail
commit()
Commits the route of this edge to the corresponding Link, if any. By default this does nothing.
getOtherVertex(v)
Returns the edge's vertex at the other of this edge from the given vertex.
Parameters:
